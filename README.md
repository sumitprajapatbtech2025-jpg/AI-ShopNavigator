# AI-ShopNavigator
LinkGenius-AI is an advanced AI-powered product intelligence system that generates premium product descriptions and instantly creates smart purchase links. It intelligently redirects users to platforms like Amazon, Flipkart, or Google Search, enabling seamless and faster buying decisions.
# ✦ LinkGenius-AI

## 🚀 Overview
LinkGenius-AI is a smart AI-powered system that generates premium product descriptions and provides instant purchase links across Amazon, Flipkart, and Google.

## 💡 Features
- AI-generated product descriptions
- Smart shopping links
- Multi-platform redirection
- Modern UI (Galaxy Theme)
- Product scoring system

## 🛠️ Tech Stack
- Python (Streamlit)
- Groq AI (LLaMA 3.3)
- HTML/CSS

## ▶️ How to Run
1. Add your Groq API key
2. Run: streamlit run app.py
3. Enter product details
4. Generate description & click to buy
